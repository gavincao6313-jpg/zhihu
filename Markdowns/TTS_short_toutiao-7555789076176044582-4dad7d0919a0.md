# Chrome DevTools MCP 工具安装与使用演示

## 1. 内容概览
本视频介绍了名为 "Chrome DevTools (MCP) for your AI agent" 的工具，该工具旨在为 AI 编程助手（如 Claude Code, Cursor, Trae, Qoder, CodeBuddy）提供实时监测浏览器的能力。通过安装此 MCP 工具，开发者无需在浏览器和 AI 工具之间手动切换反馈，AI 即可自动执行性能追踪、瓶颈分析、网络吞吐量分析及截图记录等操作。视频详细演示了如何复制配置代码并在 IDE（以 Coder/Cursor 为例）中完成 MCP 服务的添加与连接。
