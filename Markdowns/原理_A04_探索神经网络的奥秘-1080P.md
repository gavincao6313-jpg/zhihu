## 2. 核心知识字典（Glossary）
- **模型 (Model):** 视频中指代能够解决实际问题的数学公式。它接收输入数据，经过一系列计算后，输出符合预期格式的结果。
- **分类问题 (Classification Problem):** 最常见的模型任务之一，本质上是“N选1”的问题，即从N个预设类别中选择一个最符合输入数据的类别。
- **参数 (Parameters):** 模型中可学习的变量，例如线性公式 `y = ax + b` 中的 `a` (斜率) 和 `b` (截距)。通过训练数据调整这些参数，使模型能够更好地拟合数据并解决问题。
- **神经网络 (Neural Network):** 一种模拟人脑神经元结构的“万金油公式”，旨在设计一种能够通过学习适应各种复杂问题的通用公式结构。

## 2. 核心知识字典（Glossary）
- **像素值 (Pixel Value):** 图像中每个点（像素）的亮度或颜色强度，通常在 0 到 255 之间表示（0 为黑色，255 为白色）。
- **向量 (Vector):** 在机器学习中，用于表示数据特征的有序数值列表。例如，一个图像可以被展平为一个高维向量，其中每个元素代表一个像素值。
- **神经网络 (Neural Network):** 一种模拟人脑神经元连接方式的计算模型，通过多层节点（神经元）之间的连接和权重调整来学习和处理数据，实现模式识别、分类等任务。
- **参数 (Parameters):** 神经网络模型中的可调数值（如权重 `w` 和偏置 `b`），它们在训练过程中通过学习数据来确定，以优化模型的性能。
- **边缘提取 (Edge Extraction):** 一种图像处理技术，通过计算相邻像素之间的差异来识别图像中物体的边界或轮廓，是图像识别中的重要特征。

## 2. 核心知识字典（Glossary）
- **梯度下降 (Gradient Descent)：** 一种优化算法，通过计算损失函数对参数的导数，确定参数变化的方向（变大或变小），从而以最快路径寻找损失函数的最小值。
- **步长 (Step Size / Learning Rate)：** 在梯度下降中，参数每次更新的幅度。步长过大可能导致跳过最小值（跑飞），步长过小则导致训练速度极慢。
- **模型收敛 (Convergence)：** 指在训练过程中，随着迭代次数增加，损失函数的值逐渐减小并趋于稳定，不再有显著下降的状态。
- **对数损失 (Log Loss)：** 常用于概率输出模型的损失函数。它利用对数函数的特性，对预测概率与真实标签之间的偏差进行惩罚，预测越离谱，惩罚（误差值）越大。

### [02:45 - 07:30] 梯度下降的直观理解：偏导数与“切一刀”
- **核心论点：** 通过固定其他参数并观察单一参数的变化，可以确定优化方向。
- **详细展开：**
    - **梯度下降的本质：** 对于拥有成千上万个参数（如 LLM 中的 $W$ 参数）的模型，原理与 $a, b$ 相同：确定每个参数应该变大还是变小。
    - **数学操作（偏导数直观化）：** 
        - 假设固定 $b = -5$ 不变，在三维曲面上沿着 $b=-5$ 的位置“切一刀”，会得到一条关于 $a$ 的二维抛物线（微笑曲线）。
        - 在当前点（如 $a=3$），观察曲线斜率。如果向右移动（$a$ 变大）误差下降，则确定 $a$ 应该变大。
        - 同理，固定 $a$ 切换到 $b$ 的维度，确定 $b$ 的变化方向。
    - **LLM 类比：** LLM 的训练也是在定义一个极其复杂的总误差函数，然后计算几千亿个参数中每一个的梯度方向。
- **重要金句/原话：** “所谓模型训练、模型学习、机器学习，就是要找到一组优秀的 $a$ 和 $b$，想办法让总误差最小化。”

- **详细展开：**

    - **对数函数图像（$y = \ln x$）：**

        - 当 $x$ 趋近于 0（预测非常离谱）时，$y$ 趋近于负无穷。
    - **对数损失的应用：** 
        - 在计算误差时，取负对数（$-\ln x$）。
        - **案例：** 识别数字“3”。如果模型给出的“3”的概率是 0.91，误差极小；如果概率是 0.41，误差会陡增。
