# 90%的人都在错误使用AI：两个方法避开同质化，输出高质量内容

## 1. 内容概览
本视频指出90%的用户在使用AI时存在误区，即直接给AI贴上“顶级专家”、“资深导演”等标签，导致输出结果模式化、无风格。主讲人提出了两个核心改进方法：
1. **具体人物/风格映射**：不直接定义角色，而是让AI推荐该领域的真实大佬（附带代表作和擅长技能），用户选择一位后，让AI模仿其思维方式和风格进行创作（如诺兰的非线性叙事）。
2. **明确用户画像与背景**：告诉AI“你是谁”，包括身份、工具、受众画像（年龄、性别比例）、目标及过往经历。通过提供详尽的背景信息，使AI输出量身定制的方案，而非通用模板。

## 2. 时间线

### [00:00:00 - 00:01:00] 常见误区：贴标签导致同质化
讲解直接称呼AI为“顶级专家”或“爆款文案高手”的弊端。这种做法会让AI匹配最通用的模板化人设，导致输出内容烂大街、没风格。提出将教授两个有效方法来避免同质化。

### [00:01:00 - 00:02:00] 方法一：指定领域内的真实大佬
阐述如何通过让AI推荐三个具体领域的真实专家（含姓名、代表作、技能）来解决问题。以AI视频剧本创作为例，对比直接要求写剧本（三幕式教科书结构）与仿照特定导演（如克里斯托弗·诺兰的非线性叙事、高概念，或昆汀的多类型融合）的区别。强调这种方法能带来独特的个人特点和电影风格。

### [00:02:00 - 00:03:00] 方法二：明确“你是谁”（用户背景注入）
指出90%的人忽略了告诉AI自己的身份。详细列举了需要提供的信息维度：身份（如AI短视频博主）、使用的工具（如J.A., 申图生视频）、受众画像（年龄段、女性比例、偏好内容）。同时给出了其他领域的例子，如学习规划（过往工作、技能、每日时长、大厂目标岗位）和跨境电商选品（平台、客单价、风格、受众男女比例）。

### [00:03:00 - 00:03:21] 总结与资源领取
总结核心逻辑：AI是员工，必须向其传达老板的所有信息及期望达成的目标。最后提及整理了10多个不同行业的具体用法模板，引导观众在评论区留言领取。

## 3. 关键事实
*   **痛点**：直接使用“你是顶级专家”等提示词，AI会输出最大公约数的平庸内容。
*   **技巧1（外部参照）**：Prompt策略应为“请推荐三个[领域]的真实大佬，附带姓名、代表作、擅长技能”，然后选择一人进行风格模仿。
    *   *案例*：科幻片剧本 -> 选择诺兰 -> 生成非线性叙事+高概念剧本。
*   **技巧2（内部画像）**：必须向AI披露自身背景信息。
    *   *包含要素*：身份、工具链、受众数据（年龄/性别/偏好）、过往经历、可用时间、具体目标。
*   **金句**：“AI就是你的员工，想让员工干明白活儿，就先告诉他你这个老板所有的信息以及你期望最终达成的目标到底是什么。”

## 4. 可检索细节

*   **正确Prompt步骤2**：“我选择[名字]，请仿照他的思维/风格来写[任务]。”

    *   **导演**：诺兰（Christopher Nolan）-> 关键词：非线性叙事、高概念。
