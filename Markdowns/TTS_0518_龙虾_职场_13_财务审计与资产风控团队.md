# 龙虾_职场_13_财务审计与资产风控团队

## 1. 视频元数据
- **推测主题：** 本视频详细介绍了如何使用OpenCloud平台构建一个财务审计与资产风控系统，通过多Agent协作实现发票识别、财务核对、风险预警和预算分析的自动化，从而提升效率、降低成本并增强合规性。
- **核心关键词：** OpenCloud, 财务审计, 资产风控, Agent, 发票识别, 预算分析, 风险预警, 飞书多维表格, 自动化, 数字化
- **适用受众/场景：** 适用于需要自动化财务审计流程、提升财务管理效率、降低人工错误和合规风险的企业，特别是财务部门、审计团队以及希望利用AI技术进行数字化转型的组织。

## 2. 核心知识字典（Glossary）
- **OpenCloud**: 一个用于构建和部署AI Agent的平台，强调数字化、流程化和信息流处理能力。
- **Main Agent (主Agent)**: 财务审计系统的总控层，负责任务分发、流程协调、结果聚合和用户交互，不直接执行具体业务逻辑，而是协调子Agent完成任务。
- **子Agent (Sub-Agent)**: 专注于特定财务审计任务的专业Agent，包括票据识别、财务核对、风险预警和预算分析。
- **飞书多维表格**: 在本系统中作为数据存储和集成层，用于存储员工预算、报销记录和预警记录等数据，尤其适用于没有大型OA系统的团队。
- **发票查验平台**: 国家税务总局提供的在线发票查验系统，用于验证增值税发票的真伪和合规性。
