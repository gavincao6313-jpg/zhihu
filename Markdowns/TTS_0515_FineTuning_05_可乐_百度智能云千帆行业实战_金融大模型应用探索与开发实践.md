## 2. 核心知识字典（Glossary）

- **通用大模型 (General Large Language Model)**: 指在海量通用文本数据上进行预训练，具备广泛语言理解和信息处理能力的基础模型，但可能在特定行业（如金融）的专业知识和复杂任务处理上存在短板。
- **金融大模型 (Financial Large Language Model)**: 针对金融行业的特殊性，通过增量预训练、指令微调和强化学习等方式，在通用大模型基础上注入金融专业知识、提升金融领域特定能力并对齐行业偏好，以更好地服务金融业务场景。
- **百度智能云千帆平台 (Baidu Smart Cloud Qianfan Platform)**: 百度智能云推出的一站式大模型开发平台，为个人和企业开发者提供数据管理、模型训练、评估预测、服务部署、Prompt工程等完整工具链，并支持多种预置模型和数据。
- **增量预训练 (Incremental Pre-training)**: 在通用大模型的基础上，通过加入大量金融行业专业语料（如书籍、百科、公告等）进行二次训练，以提升模型对金融领域知识的掌握程度。
- **指令微调 (Instruction Fine-tuning / SFT)**: 在预训练模型的基础上，通过提供或生成丰富的金融指令数据（Prompt-Response对）进行有监督训练，以激活和提升大模型在金融任务上的对话、生成和解决问题能力。
- **强化学习 (Reinforcement Learning / RHF)**: 通过奖励模型（Reward Model）学习人类在语言、安全、金融行业上的价值偏好，并利用强化学习算法（如HF、DPO）对大模型进行对齐，以解决幻觉、不准确、遗忘等问题，使模型输出更符合人类预期和行业标准。
