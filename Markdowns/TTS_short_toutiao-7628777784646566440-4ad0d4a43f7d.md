# 给朋友们挖了个宝藏级的开源项目，用AI编程工具做出来的UI啊，怎么看都有十足的R它是把全球网站做的好在大厂的UI做成了一个设计规范

## 1. 内容概览

本视频介绍了一个名为 `Awesome-design` 的开源项目，该项目收集了全球优秀网站和大厂的 UI 设计规范。作者演示了如何将该规范导入 AI 编程工具（以 Trae 3.0 为例），从而让 AI 生成的 UI 符合专业设计标准，摆脱“AI 味”。视频详细展示了从下载项目、配置提示词到最终生成高质量 UI 的全过程。

## 2. 时间线

### [00:00:00 - 00:01:00] 项目介绍与准备

作者推荐 `Awesome-design` 开源项目，称其将全球优秀 UI 整合为设计规范。步骤包括：
1.  在 GitHub 搜索并下载项目压缩包。
2.  新建项目目录并解压，内含 59 个代表性站点的设计文档。
3.  复制解压后的绝对路径备用。
4.  打开 AI 工具 Trae 3.0，切换到 Code 模式，点击“应用开发”。

### [00:01:00 - 00:01:55] 提示词配置与执行

1.  **关键提示词**：在默认提示词基础上，必须添加以下指令以强制 AI 使用本地规范：
    *   “请你在开发前深入分析。”
    *   粘贴之前复制的文件夹路径：“这里我们粘贴我们刚复制备用的那个路径，这个文件夹中的所有文档。”
    *   “你在开发设计UI时候呢，必须遵循该文档中的设计规范和要求，不得有任何偏离进复述我的需求，我确认后再执行。”
2.  **交互流程**：AI 会先分析文档，然后询问用户偏好哪种风格。
3.  **结果对比**：使用规范后的 UI 更具“人味”和专业感，而非典型的 AI 生成样式。
