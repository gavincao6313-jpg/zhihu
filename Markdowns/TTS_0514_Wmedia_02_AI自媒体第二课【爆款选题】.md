# Wmedia_02_AI自媒体第二课【爆款选题】

## 1. 视频元数据
- **推测主题：** 本视频详细讲解了如何利用AI编程工具，为自媒体创作者生成爆款选题和标题，并深入剖析了自媒体内容创作的核心认知、算法机制及产品开发方法论。
- **核心关键词：** AI编程, 自媒体, 爆款选题, 标题生成, 提示词工程, 内容创作, 算法机制, 小红书, MVP原则, 个性化内容配置
- **适用受众/场景：** 适用于自媒体小白、初级创作者，以及希望通过AI工具提升内容创作效率、解决选题和标题痛点的个人或团队。

## 2. 核心知识字典（Glossary）

-   **爆款选题 (Trending Topic/Viral Topic):** 指能够引发广泛关注、高互动、高流量的内容主题。其核心在于解决用户痛点、提供价值，且具有可重复、可循环、有周期的特点。
-   **标题 (Title):** 内容的“广告”，旨在吸引眼球、制造好奇，引导用户点击和观看。它不只是内容的概括，更需要运用营销思维和心理学，包含强相关、高价值、高情绪和低门槛的元素。
-   **MVP (Minimum Viable Product - 最小可行性产品):** 在AI编程和产品开发中，指只保留核心功能、能够跑通最小可行性流程的产品版本，避免一开始就追求大而全，以降低开发难度和提高成功率。
-   **System Prompt (系统提示词):** 在AI交互中，用于定义AI的身份、角色和基本规则的提示词，例如“你是一个擅长为自媒体创作者赋能的CTO”。
-   **User Prompt (用户提示词):** 在AI交互中，用户向AI提出的具体需求和指令，例如“帮我生成美妆行业的短视频选题，时长一分钟”。
-   **流量激勵機制 (Traffic Incentive Mechanism):** 平台根据用户对内容的互动行为（如点赞、收藏、评论、转发、关注）赋予不同权重，以此判断内容质量并进行流量分发的机制。

### [00:16:02 - 00:24:02] 自媒体核心认知：选题与标题的重要性及区别

-   **核心论点：** 选题和标题是自媒体内容成功的关键，它们虽然紧密相关，但本质、目标和底层思维完全不同。选题决定内容本身和受众量级，标题是内容的“广告”和“钩子”。
-   **详细展开：**
    1.  **自媒体现状与“卷”的极致：** 讲师指出，当前自媒体行业竞争激烈，80%的创作者只获得20%的流量，无论是抖音还是小红书，尾部创作者占比都很大。小红书相对对素人友好，有去中心化特点，零粉也有机会爆款。在这种“卷”到极致的环境下，唯一的出路是“用内容撞平台的算法机制”。
    2.  **小红书算法机制举例：** 讲师以小红书为例，解释了流量激励机制（CES）。
        *   CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8。
        *   关注的权重最高（8分），其次是评论和转发（4分），点赞和收藏最低（1分）。
        *   分级流量池：0-100 -> 200-500 -> 3000-5000 -> 1w-2w -> 10w-20w -> 100w-300w。

### [01:27:27 - 01:31:45] 说人话AI编程小技巧

-   **核心论点：** 即使不懂专业术语，也能通过“大白话”清晰地表达需求，从而成功制作AI工具。关键在于清晰地知道、表达和描述自己想要什么。
-   **详细展开：**
    1.  **专业术语的困惑：** 许多非代码、非计算机专业的学员在AI编程中会遇到专业术语（如JSON格式、System Prompt、User Prompt）的困惑。
    2.  **“大白话”解释与对比：**
        *   **JSON格式：**
            *   专业：一种数据交换格式，使用键值对表示数据。
            *   大白话：用固定格式把信息包起来，方便电脑读取。
            *   区别：不用JSON可能输出一大段难以提取的文本，内容位置不固定；用JSON则程序能轻松读取，想要哪个直接抓哪个。
        *   **System Prompt vs User Prompt：**
            *   System Prompt（系统提示词）：定义AI身份和规则。例如：“你是一个爆款选题专家。你的规矩是：1. 一次给5个选题。2. 每个选题要有标题、说明、理由。3. 按JSON格式返回。”
            *   User Prompt（用户提示词）：具体要什么。例如：“帮我生成选题：- 行业：美妆 - 时长：1分钟。”
    3.  **专业词 vs 大白话，哪个强？**
        *   讲师通过对比“不要包含额外文本” vs “不要‘好的，我来帮你’这些废话”，以及“生成5个选题” vs “一次性给我5个选题”，发现大白话并不影响结果。
    4.  **结论：** 大白话不影响结果，只要能做到三件事：**清晰地知道、表达、描述自己要什么**。
    5.  **“大白话作弊清单”：** 讲师提供了一个清单，将常见AI编程术语翻译成大白话，方便学员查阅和使用。

### [01:31:45 - 01:37:01] 学习建议与课程总结

-   **核心论点：** 学习AI编程应注重实践，并以制作选题计划为终极目标，通过短平快制作解决内容焦虑，实现内容裂变。

        *   **自媒体基础认知：** 内容撞平台算法、选题定生死、标题是钩子、选题计划。
