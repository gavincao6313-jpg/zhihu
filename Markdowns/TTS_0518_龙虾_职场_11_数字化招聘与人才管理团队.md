# 龙虾_职场_11_数字化招聘与人才管理团队

## 1. 视频元数据
- **推测主题：** 本视频详细介绍了如何利用 OpenClaw 平台构建一个数字化的招聘与人才管理团队，通过多 Agent 智能协作实现招聘流程自动化、效率提升和成本降低。
- **核心关键词：** OpenClaw, 数字化招聘, 人才管理, Agent 智能协作, 简历解析, 面试协调, 会议助理, 对话管理, 自动化流程, 飞书集成, 邮件协调, AI 面试
- **适用受众/场景：** 适用于希望通过 AI 和自动化技术优化招聘流程、降低人工成本、提升招聘效率和人才匹配度的企业HR、招聘团队以及技术开发者。

## 2. 核心知识字典（Glossary）
- **Agent (智能体):** 在 OpenClaw 平台中，Agent 是具备特定职能的自动化实体，可以接收任务、执行操作并与其他 Agent 协作。视频中强调了主 Agent 负责调度，专业化子 Agent 负责具体任务。
- **OpenClaw:** 一个处理信息流和多 Agent 之间协作的平台，在数字化招聘和人才管理场景中具有优势，能够实现统一化的数据处理和流程自动化。
- **简历解析 Agent (Resume Parser Agent):** 专门负责从各种格式（如MD、Word）的简历中提取关键信息，如姓名、联系方式、教育背景、工作经验、技能等，并将其结构化存储到人才库中。
- **招聘协调 Agent (Recruitment Coordinator Agent):** 负责根据招聘任务和人才库信息，匹配合适的候选人与面试官，并协调面试流程，包括发送面试邀请、时间协调等。
- **对话管理 Agent (Dialogue Management Agent):** 负责与候选人和面试官进行多轮沟通，处理邮件回复（接受、拒绝、请求更改时间、不明确回复），确保沟通顺畅和高效。
- **会议助理 Agent (Meeting Assistant Agent):** 负责创建和管理面试会议，例如在飞书上创建会议，并向参与者发送会议链接和通知。
