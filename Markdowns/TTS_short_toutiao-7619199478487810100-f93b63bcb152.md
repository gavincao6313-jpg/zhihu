# GitHub一周7个神级AI项目：从全自动科研到自我进化技能库

## 1. 内容概览
本视频盘点了过去两周GitHub上诞生的7个热门AI开源项目。这些项目涵盖了AI代理组建、全自动科研、无头浏览器、本地模型运行检测、多工具AI控制、高性能数据库以及自我进化技能库等领域。视频逐一介绍了每个项目的核心功能和亮点，强调了它们对AI玩法的颠覆性影响，如零成本雇佣AI公司、睡一觉醒来获得最优模型版本、AI可控的所有软件等。

## 2. 时间线

### [00:00:00 - 00:01:00] 项目1-3：Agency Agents, AutoResearch, Light Panda
1.  **Agency Agents**: 一键组建51位AI专家，覆盖产品开发、设计、营销等9个部门，相当于零成本雇佣一家AI公司。
2.  **AutoResearch**: AI大神Karpathy开源的全自动科研实习生。给定目标后，自动规划实验、改代码、跑训练、看结果，最终输出最优版本。
3.  **Light Panda**: 专为AI打造的无头浏览器，不依赖Chrome，从零构建。速度快11倍，内存少9倍，适合数据爬取和AI智能体开发。

### [00:01:00 - 00:01:50] 项目4-7：Local Model Runner, Anything, Spacetime DB, Cognee
4.  **Local Model Runner**: 一条命令检测哪些模型能在本地电脑运行，并对质量和速度打分，避免下载大模型后跑不动的尴尬。
5.  **Anything**: 港大团队开源，一行命令让修图、3D软件、OBS录屏等变成AI可控工具。
6.  **Spacetime DB 2.0**: 国外抢手的高性能数据库，声称比传统数据库快1000倍。将数据库和服务器合二为一，客户端直连。
7.  **Cognee**: 给AI装上能自我进化的技能库。执行失败时，自动查找原因、修改代码、优化技能，使AI技能从静态文件变为活的升级系统。
