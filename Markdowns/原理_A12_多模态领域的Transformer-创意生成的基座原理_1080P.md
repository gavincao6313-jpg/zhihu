## 2. 核心知识字典（Glossary）

- **多模态模型 (Multimodal Model):** 指输入或输出不只包含一种模态（如文本、图像、音频等）的模型。例如，输入是文本和图像，输出是文本。
- **视觉模型 (Visual Model):** 专门处理图像、视频等视觉信息的模型。在视频中，特指纯粹以图像作为输入和输出的模型。
- **推理模型 (Reasoning Model):** 指具备推理能力的模型，如DeepSeek R1和O1，它们在特定任务上通过训练的思维过程进行推理，而非仅仅是简单的模式识别。
- **卷积核 (Convolutional Kernel):** 卷积神经网络中的核心组件，是一个小的矩阵，通过与输入图像的局部区域进行卷积运算，提取图像中的特征。卷积核中的每个数字都是神经网络中待确定的参数。

- **核心论点：** 卷积核通过与输入图像的局部区域进行加权求和运算，生成输出图像的对应像素值，其内部参数是神经网络中待确定的。

## 2. 核心知识字典（Glossary）

*   **Residual Network (ResNet):** 一种深度神经网络架构，通过引入“残差连接”（或跳跃连接）来解决深度网络训练中的梯度消失/爆炸问题和性能退化问题。它允许层学习残差函数 `F(x)` 而不是直接的映射 `H(x)`，输出为 `F(x) + x`。
*   **过拟合 (Overfitting):** 机器学习模型在训练数据上表现过好，以至于它学习了训练数据中的噪声和不相关特征，导致在未见过的新数据上性能下降。
*   **梯度消失/梯度爆炸 (Vanishing/Exploding Gradients):** 深度神经网络训练中的常见问题。梯度在反向传播过程中变得过小（消失）或过大（爆炸），导致网络权重更新困难，模型难以有效学习。
*   **Vision Transformer (ViT):** 一种基于Transformer架构的图像识别模型。它将图像分割成固定大小的图像块（patches），将每个patch线性嵌入，并添加位置编码，然后将这些序列化的向量输入到标准的Transformer编码器中进行处理。
*   **Patch (图像块):** 在Vision Transformer中，图像被分割成的小的、固定大小的区域，每个patch被视为一个独立的输入单元（类似于自然语言处理中的“token”）。

### [00:06:40 - 00:07:24] ResNet的核心思想：残差连接（Residual Connection）

- **核心论点：** ResNet通过引入残差连接，允许网络层学习残差函数，从而更容易训练深度网络并避免性能退化。

- **核心论点：** ViT通过将图像分割成固定大小的图像块（patches），并将其线性嵌入和序列化，从而将图像数据转换为Transformer可以处理的序列格式。

## 2. 核心知识字典（Glossary）

*   **Vision Transformer (ViT):** 一种将 Transformer 架构应用于图像识别的模型。它将图像分割成固定大小的图像块（patches），然后将这些块作为序列输入 Transformer Encoder 进行处理，实现图像分类。
*   **Swin Transformer:** 一种分层视觉 Transformer，通过引入 Shifted Windows（移位窗口）机制来解决传统 ViT 在处理不同尺度视觉实体和高分辨率像素时的挑战。它构建了分层的特征图，并允许在非重叠的局部窗口内进行自注意力计算，同时通过跨窗口连接实现全局信息交互，具有线性计算复杂度。
*   **CLIP (Contrastive Language-Image Pre-training):** 由 OpenAI 提出的多模态模型，旨在连接自然语言和图像。它通过对比学习（Contrastive Learning）预训练一个文本编码器和一个图像编码器，使其能够理解图像和文本之间的语义关联，从而实现零样本（Zero-Shot）预测和多模态任务。
*   **Shifted Windows (移位窗口):** Swin Transformer 中的核心机制，用于在 Transformer 层之间交替地对图像块进行分区，以实现跨窗口的信息交互，同时保持计算效率。
*   **Contrastive Pre-training (对比预训练):** CLIP 采用的一种预训练方法，通过最大化匹配的文本-图像对的相似度，同时最小化不匹配对的相似度，来学习图像和文本的联合表示。

### [03:45:00 - 05:30:00] Swin Transformer 的 Shifted Window 机制详解

*   **核心论点：** Swin Transformer 通过 Shifted Window 机制，在不同层级上动态调整自注意力计算的窗口，既考虑了图像的尺度特性，又避免了全局自注意力带来的高计算成本，同时完全不依赖 CNN 结构。
*   **详细展开：**
    *   Swin Transformer 的核心改进在于其分层结构和 Shifted Window 机制。
    *   在第一层，它将图像划分为多个小的红色方框（例如 4x4），每个方框内有 16 个图像块。自注意力计算只在这些局部方框内进行，保证了局部信息的聚合。
    *   随着层级的加深，方框的大小逐渐增大（例如 8x8，然后 16x16），但每个方框内包含的图像块数量（16个）保持不变。这使得模型能够捕获不同尺度的特征。
    *   "Shifted Window" 的含义是，在相邻的 Transformer 层之间，窗口的划分会进行偏移。
        *   例如，在 Layer L，窗口是规则划分的。
        *   在 Layer L+1，窗口会进行偏移，使得原本在不同窗口中的图像块现在可以在同一个窗口中进行自注意力计算。
        *   这样，通过两轮（或更多轮）的偏移，每个图像块都能间接地与更广范围的图像块进行信息交互，从而实现了全局感受野，同时避免了全局自注意力的高计算量。
    *   讲者强调，Swin Transformer 完全没有使用任何卷积神经网络（CNN）的结构，它仍然是纯粹的 Transformer 模型，但通过巧妙的窗口设计解决了视觉任务中的尺度问题和计算效率问题。
*   **视觉/屏幕内容：**

        *   "Layer l+1" (right): 窗口进行偏移，使得原本在不同窗口中的图像块现在可以在同一个窗口中进行自注意力计算（红色圆圈）。

*   **重要金句/原话：** "我的自注意力机制啊，我不是全局去做，我觉得不需要全局去做自注意力机制...我只需要在某一个小区里面去做自注意力机制...我画了很多这种小的红格子。我的自注意力机制只在这个小红格里，每一个小红格子里面呢，又有很多的Patch...我保证我的每一个小红方框里面呢，有16个Patch。这16个Patch之间是可以交换数据，做多头自注意力机制的。" "它在每一层的时候，它先这样作一次自注意力机制...再这样作一次...这样作两轮自注意力机制之后，才算是做完一层Transformer。" "它这个不停的在变化的窗口叫做一个，就是在游走的，一个可以变化的Window，它就叫做Swin。" "它都是直接提升两个点或者直接提升三个点，然后呢，就不像别人那种只是提升0.5个点，或者是提升一个点。"

### [06:00:00 - 07:00:00] CLIP 的工作原理：对比学习与零样本预测

*   **核心论点：** CLIP 通过大规模的文本-图像对进行对比学习，训练出能够将图像和文本映射到同一嵌入空间的编码器，从而实现强大的零样本（Zero-Shot）预测能力。
*   **详细展开：**
    *   CLIP 的模型结构包含两个核心模块：一个 Text Encoder（文本编码器）和一个 Image Encoder（图像编码器）。
    *   Text Encoder 实际上是一个 Transformer Encoder，用于将任意一段文本转化为一个向量（Embedding）。
    *   Image Encoder 实际上是一个 ViT（Vision Transformer），用于将任意一张图片转化为一个向量。
    *   **训练数据：** OpenAI 收集了 4 亿组文本-图像对数据。这些数据来源于互联网，例如百度图片、Google 图片、Instagram 等，每张图片旁边通常都有描述性文字。
    *   **对比学习（Contrastive Pre-training）：**
        *   假设有 100 段文本和 100 张图片，它们是一一对应的（第一段文本描述第一张图片，以此类推）。
        *   将这 100 段文本输入 Text Encoder，得到 100 个文本向量（T1, T2... T100）。
        *   将这 100 张图片输入 Image Encoder，得到 100 个图像向量（I1, I2... I100）。
        *   然后，计算所有文本向量和图像向量之间的相似度（使用点积）。
        *   目标是：对角线上的相似度（Ti 与 Ii 之间的相似度）应该尽可能高，因为它们是匹配的文本-图像对。
        *   非对角线上的相似度（Ti 与 Ij (i≠j) 之间的相似度）应该尽可能低，因为它们是不匹配的文本-图像对。
        *   通过定义总误差（对角线相似度之和减去非对角线相似度之和），并最小化这个误差，来训练 Text Encoder 和 Image Encoder 的参数。
        *   初始时，两个编码器都是随机的，输出的向量很"笨"，相似度计算结果也不准确。但随着训练的进行，它们会变得越来越"聪明"。

    *   图示 "Overview A: Contrastive pre-training"。
        *   左上角 "pepper the aussie pup" (文本) 输入 "Text Encoder"，输出 T1, T2... TN (文本向量)。
        *   左下角一张狗的图片输入 "Image Encoder"，输出 I1, I2... IM (图像向量)。

### [07:00:00 - 07:50:00] CLIP 的零样本预测（Zero-Shot Prediction）

*   **核心论点：** 经过对比学习训练后，CLIP 模型可以利用提示词（Prompt）进行零样本预测，即在没有见过特定类别训练数据的情况下，也能对新类别进行分类。
*   **详细展开：**
    *   训练结束后，CLIP 得到了两个强大的编码器：Text Encoder 和 Image Encoder。
    *   **零样本预测过程：**
        1.  **准备提示词：** 对于要分类的每个类别（例如 plane, car, dog, bird），构造一个提示词，例如 "a photo of a plane"（一张飞机的照片）。
        2.  **编码提示词：** 将所有这些提示词输入 Text Encoder，得到每个类别的文本向量（T1, T2... TN）。
        3.  **编码待预测图像：** 将待预测的图像（例如一张狗的图片）输入 Image Encoder，得到一个图像向量（I_query）。
        4.  **计算相似度：** 计算 I_query 与所有类别文本向量（T1, T2... TN）之间的相似度。
        5.  **预测：** 图像向量与哪个类别文本向量的相似度最高，就将该图像归为那个类别。
    *   这种方法使得 CLIP 具有强大的泛化能力，可以处理开放世界中的任意视觉概念，而无需为每个新任务重新训练模型。

## 2. 核心知识字典（Glossary）

*   **CLIP (Contrastive Language-Image Pre-training):** 一种由OpenAI提出的多模态模型，通过对比学习将图像和文本映射到同一个嵌入空间，从而实现零样本图像分类、图像检索等任务。它使用独立的图像编码器和文本编码器。
*   **VILT (Vision-and-Language Transformer):** 一种多模态Transformer模型，旨在通过使用单个Transformer编码器统一处理图像和文本输入，以提高效率和简化架构，实现视觉和语言的联合理解。
*   **零样本预测 (Zero-shot Prediction):** 指模型在训练时未见过特定类别的样本，但在推理时能够对这些新类别进行准确分类的能力。CLIP通过将类别标签转换为自然语言描述来实现零样本预测。
*   **稠密架构 (Dense Architecture):** 指Transformer模型中所有参数在每次推理或每个token的计算中都会被激活和使用，相对于稀疏激活的模型（如MoE）而言。
*   **MoE (Mixture-of-Experts) 架构:** 一种Transformer架构，其中模型包含多个“专家”网络。在推理时，一个门控网络会根据输入动态选择并激活其中一个或几个专家网络，从而实现稀疏激活，提高计算效率，尤其适用于超大规模模型。

## 2. 核心知识字典（Glossary）
- **基座模型 (Foundation Model)：** 指在大量数据上预训练的通用模型，能够适应广泛的下游任务，通过微调或少量样本即可在多个领域取得良好性能，大幅降低开发成本。
- **CLIP (Contrastive Language-Image Pre-training)：** 一种由 OpenAI 开发的多模态模型，通过对比学习将图像和文本嵌入到同一个语义空间中，实现图像-文本的相互理解和检索。
- **FP8 (8-bit Floating Point)：** 一种低精度浮点数格式，用于模型训练和推理，旨在减少内存占用和计算量，以提高效率。
- **层归一化 (Layer Normalization)：** 深度学习中用于稳定神经网络训练的技术，尤其在残差连接中，通过规范化层内激活的均值和方差，防止梯度爆炸或消失。
