# Obsidian上手第一天必做设置：同步、插件与AI火力全开指南

## 1. 内容概览
本视频为Obsidian新手入门指南，旨在帮助用户在第一天避开常见坑点，快速配置高效笔记工作流。内容涵盖：多端同步方案选择与设置（以iCloud为例）、基础设置（中文界面、附件管理）、核心插件启用（反向链接、属性列表）、笔记属性与数据库管理、以及AI集成配置（Tree/Claude Code与Obsidian插件）。目标是让用户第一天就能接上AI火力，实现流畅的知识管理。

## 2. 时间线

### [00:00:00 - 00:01:00] 引言：Obsidian的门槛与AI潜力
指出Obsidian高度自由带来的上手门槛，承诺通过基础设置解决痛点。介绍作者身份：Blink AI+知识管理实践者。目标：第一天配置好，接上AI火力。

### [00:01:00 - 00:02:00] 第一步：同步方案选择与iCloud实操
同步方案推荐：
1. 苹果全家桶：iCloud（无缝、免费）。
2. 官方同步：全平台支持，$10/月。
3. 国外非苹果：Google Drive/Dropbox。
4. 国内非苹果：微软OneDrive。
iCloud实操：iPhone创建仓库 -> Mac打开本地仓库 -> 右键iCloud文件夹开启“保留下载”（避免反复在线下载影响效率）。

### [00:02:00 - 00:04:00] 基础设置：语言、附件管理与核心插件
基础设置：
1. 语言切换为简体中文（需重启）。
2. 文件与链接：开启“时钟更新内部链接”。
3. 附件管理：设置统一附件文件夹（如`attachments`），避免根目录混乱，方便迁移。
核心插件开启：
1. 反向链接（Backlinks）。
2. 命令面板（Command Palette）。
3. 属性列表（Properties List）。

### [00:04:00 - 00:06:00] 核心功能演示：反向链接与笔记属性
反向链接：输入`[[`创建双向链接，右侧面板显示引用笔记及具体引用位置，支持局部关系图。
笔记属性：
1. 自动属性：创建时间、更新时间。
2. 自定义属性：类型（Card Note）、标签（纳瓦尔、知识管理）、出处（播客）。
3. 项目管理：添加“状态”属性（未开始/进行中/已完成），模拟Notion数据库管理。

### [00:06:00 - 00:07:49] 高级功能：Dataview与AI摘要
Dataview功能：通过表格展示筛选笔记（如今日创建的笔记、提及特定日期的笔记）。
AI摘要：集成Follow Builders Skill，自动将行业资讯同步至Obsidian日志，实现每日AI摘要。

### [00:07:49 - 00:09:38] AI配置：外部对话与Cloud Plugin
外部对话：无需插件，通过Tree等AI Agent直接读取笔记路径进行对话。
Cloud Plugin（Obsidian内嵌AI）：
1. 优势：不离App、快速感知上下文（圈选文字即可询问）。
2. 前提：已安装并配置Claude Code。
3. 安装：关闭安全模式 -> 浏览社区插件（先装一个如Excalidraw）-> GitHub下载Clodian插件文件（release版）-> 放入`.obsidian/plugins/clodian/` -> 启用。
4. 配置：设置称呼、附件路径（如`attachments`）、自定义变量。

### [00:09:38 - 00:12:25] Cloud Plugin安装细节与使用
详细步骤：
1. GitHub搜索`clodian`，下载Release中的三个文件。
2. Mac访达中显示隐藏文件（Cmd+Shift+.）。
3. 进入`.obsidian/plugins/`，新建`clodian`文件夹，放入文件。
4. 在Obsidian设置中启用插件，配置语言和附件路径。
5. 左侧工具栏出现Cloud图标，右侧打开对话面板。
结语：鼓励上手实践，后续分享PARA、卡片笔记法等进阶内容。

## 3. 关键事实
*   **同步推荐**：苹果用户首选iCloud并开启“保留下载”；国内非苹果用户推荐OneDrive。
*   **附件管理**：强制统一附件路径（如`attachments`），保持笔记库整洁。
*   **核心插件**：反向链接、命令面板、属性列表。
*   **AI集成**：
    *   基础：Tree等外部Agent直接读取文件。
    *   进阶：Cloud Plugin（需Claude Code支持），实现圈选上下文对话。
*   **插件安装**：Community Plugins需关闭安全模式，部分插件（如Cloud）需从GitHub手动安装至`.obsidian/plugins/`。

*   **AI工具**：Tree, Claude Code, OpenAI, Trae。
*   **设置选项**：Settings -> Language -> Simplified Chinese; Settings -> Files & Links -> Clock update internal links; iCloud -> Keep Downloaded。
