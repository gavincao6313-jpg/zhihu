# 龙虾_职场_05_会议与日程管理团队

## 1. 视频元数据
- **推测主题：** OpenCloud 多Agent编排实现会议与日程智能管理
- **核心关键词：** OpenCloud, 多Agent, 调度器, 会议管理, 日程同步, 消息分发, 面试协调, 语音转文字, 文字转语音, 飞书日历, API-Key, Skill
- **适用受众/场景：** 开发者、企业用户、需要自动化日程和会议管理、消息通知、面试流程协调的团队。

## 2. 核心知识字典（Glossary）
- **Agent (智能体):** 在OpenCloud平台中，Agent是具备特定功能和能力的独立模块，可以执行预设的任务。本视频中，每个小功能（如日程同步、会议助手）都被封装为一个子Agent。
- **调度器 (Scheduler):** 负责统一调度和编排多个子Agent的Agent，根据用户请求智能地选择并调用合适的子Agent来完成复杂任务。
- **Skill (技能):** Agent所具备的具体能力或功能，通常通过代码或配置实现。例如，日程同步Agent的Skill包括预约会议、检查会议冲突等。
- **API-Key:** 用于身份验证和授权访问第三方服务（如电话服务）的密钥，确保只有授权用户才能调用相关功能。
- **飞书日历 (Feishu Calendar):** 飞书平台提供的日程管理工具，本案例中用于实现日程的创建、修改、提醒和冲突检测。
