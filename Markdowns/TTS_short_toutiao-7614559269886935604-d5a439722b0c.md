# Claude Code 三大插件推荐：My, Superpowers, Everything

## 1. 内容概览
视频推荐了三款能显著提升 Claude Code 生产力的插件，帮助用户从“青铜”进阶到“王者”。分别是适合团队开发的 "Oh My Claude Code" (简称 My)，注重代码质量的 "Superpowers"，以及追求极致效率的 "Everything Claude Code" (简称 Ever)。

## 2. 时间线

### [00:00:00 - 00:00:12] 插件一：Oh My Claude Code
介绍第一个插件，配备 7 个专业 AI 代理（规划、写代码、审核），形成“规划-执行-审核”闭环，相当于拥有一个开发团队。

### [00:00:12 - 00:00:21] 插件二：Superpowers
介绍第二个插件，被称为“代码质量神”。强制测试驱动开发 (TDD)，先写测试再写代码，自动触发机制，减少 Bug，无需手动干预。

### [00:00:21 - 00:00:29] 插件三：Everything Claude Code
介绍第三个插件，黑客松冠军配置，花费 10 个月打磨。拥有 9 个代理、11 个技能，具备本能学习系统，能自动记住用户习惯。

### [00:00:29 - 00:00:42] 总结与建议
根据用户需求给出建议：团队开发选 My，个人追求质量选 Superpowers，高级玩家要极致效率选 Ever。鼓励认真学习这些工具的代码和逻辑。
