# Spec Coding：AI编程新范式与Kiro IDE解析

## 1. 内容概览
视频介绍了AI编程范式的演变，从Tab Coding到Vibe Coding，再到最新的Spec Coding。重点解析了亚马逊云科技发布的AI IDE——Kiro，及其核心的Spec（规范）模式。通过演示“脑雾擦除器”项目，展示了Spec Coding如何通过产品需求文档、技术设计文档和任务清单，将AI从辅助工具转变为能落地复杂项目的可靠同事。

## 2. 时间线

### [00:00:00 - 00:01:00] AI编程范式演变背景
有人用AI写代码行云流水并赚钱，有人却报错连连。揭秘高手都在用的新范式：Spec Coding。回顾AI编程发展阶段：
1.  **第一阶段 Tab Coding**：卡帕西（Copilot）早期，代码股权式辅助，无限续写。
2.  **第二阶段 Vibe Coding**：Claude + Cursor结合，假装在编程的氛围感，边聊边改，适合小项目，复杂逻辑Bug难改。

### [00:01:00 - 00:02:00] 第三阶段：Spec Coding的出现
随着模型和Agent能力提升，大家要求AI做更落地、实用的开发。Spec Coding核心理念：**先出规范的计划，再干活**。
*   **早期**：手工写一堆PRD（产品需求文档）。
*   **进阶**：让AI写代码前先规划。
*   **现在**：正规的Spec模式，由Kiro IDE实现。

### [00:02:00 - 00:03:00] Kiro IDE：首个Spec模式的IDE
Kiro是亚马逊云科技全量发布的AI IDE。表面看与Cursor类似，但它是第一个拥有Spec模式的IDE。其Spec文件集合了敏捷开发、TDD（测试驱动开发）、Copilot AI编程原则等前沿成熟范式。
**演示项目**：做一个“脑雾擦除器”（AI辅助冥想）。

### [00:03:00 - 00:04:00] Spec流程第一步：产品需求文档 (PRD)
Kiro充当产品经理角色：
1.  **核心概念与哲学**：定义产品理念（如“用户只用负责放下”）。
2.  **专业术语表**：预定义代码中的对象和类。
3.  **用户故事**：定义每个功能。
4.  **验收标准**：为每个功能设定验收标准。
确认需求后，才进入下一步。

### [00:04:00 - 00:05:00] Spec流程第二步：技术设计文档
Kiro充当架构师角色：
*   **技术选型**：选择合适的技术栈。
*   **接口设计**：定义API。
*   **数据模型设计**：数据库结构。
*   **错误处理策略**。
小白用户可将生成的Spec文档发给AI，让它解释拆解，结合执行进行学习成长。

### [00:05:00 - 00:06:00] Spec流程第三步：任务清单
Kiro充当项目经理角色：
*   **分任务**：将活拆分为基础架构、数据模型、状态管理、UI组件。
*   **迭代交付**：形成一条可以分块跑通的开发线。
点击开发后，大量时间在测试和修复，确保交付物基本能跑通。Spec文件与代码双向同步，保持一致。

### [00:06:00 - 00:07:00] Spec模式的价值与亚马逊战略
*   **价值**：包含了大厂高P级别的产品、架构、项目管理能力。是顶尖工程师的最佳实践集合。即使不做项目，学习其规范构建思路也很有价值。
*   **协作转变**：AI从给建议的工具变成承担落地的同事。人类需用科学工具、规范和流程管理AI。
*   **亚马逊战略**：Re:Invent大会核心趋势是让AI靠谱干活，从芯片、自研模型、Agent基础设施(Agent Core)到Tro等前沿Agent全覆盖。

### [00:07:00 - 00:07:20] Kiro的使用优势
*   **全量开放**：新用户注册送500 Point。
*   **国内唯一**：无门槛调用Claude的IDE。
*   **Kiro Powers**：类似Cloud Skill的AI外挂能力包，更简单。
*   **Autonomous Agent**：适合团伙作业，几小时干几天的活。

## 3. 关键事实
*   **AI编程三阶段**：Tab Coding (代码续写) -> Vibe Coding (氛围编程/边聊边改) -> Spec Coding (先规范后干活)。
*   **Kiro IDE**：亚马逊云科技发布，首个具备Spec模式的IDE。
*   **Spec三大文档**：
    1.  产品需求文档 (PRD)：产品经理视角。
    2.  技术设计文档：架构师视角。
    3.  任务清单：项目经理视角。
*   **新用户体验**：注册送500 Point，国内无门槛调用Claude。
