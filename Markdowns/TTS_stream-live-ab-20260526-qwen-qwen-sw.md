- **适用受众/场景：** AI 自媒体创业者、AI 工具开发者、希望利用 AI 提效的内容创作者、寻求职业转型的技术/非技术人员。

## 2. 核心知识字典（Glossary）
1.  **FDE (Forward Deployed Engineer / 前沿部署工程师)：** 介于产品研发与客户业务之间的角色。既懂技术边界，又深谙行业痛点与商业逻辑，负责将技术方案落地为可交付的业务价值。核心能力模型包括：技术理解力、行业洞察、沟通交付。
2.  **AI 播客三种实现路径：**
    *   **Agentic 一键生成：** 输入话题 -> 脚本 -> 音频 -> 视频。简单但不可控。
    *   **半自动流水线：** 预制素材/模板 -> 场景组装 -> 文案生成 -> 视频渲染。稳定但限制发散，易重复。
    *   **借鉴/解析对标：** 抓取爆款 -> 解析脚本 -> 改写 -> 生成。灵活，跨行业复用选题。
3.  **洗稿的核心逻辑（结构仿写 + 灵魂注入）：** 复制爆款的选题、结构和节奏（保证流量基础），但必须通过注入个人经历、踩坑故事、独特观点来增加“人格化”内容（解决原创度与高价值问题）。AI 解决 60-70 分及格线，人类提供 80-90 分金字塔尖价值。
4.  **安全底线原则：** 在自动化引流或评论回复中，**中等风险和高风险绝对不可接受**。任何导致封号、限流的方案都是负价值。对外承诺需达到 **99.9%** 的安全率（模拟真人行为、随机延迟等）。
5.  **Context Compression (上下文压缩)：** 在使用 Coze 等 LLM 平台时，避免在同一个对话窗口无限拉长上下文，否则会导致积分消耗剧增且生成质量下降。应分段对话或使用 CC 功能压缩历史记忆。
6.  **所见即所得 (WYSIWYG) 修复：** 针对视频生成工具中预览与下载结果不一致的 Bug，需通过明确 Prompt 约束画面元素的一致性，并检查后端渲染逻辑是否替换了素材。
