## 2. 核心知识字典（Glossary）
- **多维洗稿及矩阵内容团队 (OpenClaw Multi-dimensional Content Rewriting & Matrix Team):** 一个基于 OpenClaw 平台构建的数字员工团队，由多个专业 Agent 组成，旨在自动化处理原始素材、进行逻辑级洗稿、生成高质量原创内容，并为多平台矩阵账号提供发布策略和批量发布功能。
- **Agent (虾):** 在 OpenClaw 平台中，Agent 指的是具有特定功能和职责的数字员工。本视频中涉及的主 Agent 负责协调，以及素材处理、流量重组洗稿、自媒体矩阵管家等专业 Agent。
- **逻辑级洗稿:** 一种高级的洗稿方式，不仅仅是替换词语或调整句式，而是从原文中提取核心逻辑、传播基因和关键信息，然后在此基础上重新构建全新的原创内容，以避免被平台检测出洗稿痕迹，同时保留内容的传播价值。
- **矩阵账号:** 指在多个自媒体平台或同一平台的多个账号上，围绕一个核心主题或品牌，进行协同运营和内容发布的账号群。目的是扩大内容覆盖面，提升整体影响力。
- **提示词 (Prompt):** 用于指导 AI Agent 执行任务的指令或文本输入。提示词的准确性和详尽程度直接影响 AI Agent 的理解能力和输出质量。
