## 2. 核心知识字典（Glossary）
- **Agent (代理)**: 在 OpenCloud 平台中，Agent 是一个具有特定职责和能力的独立智能实体。它们可以被编排以协同完成复杂任务，例如调度、热点捕捉、内容改写和消息分发。
- **多 Agent 编排 (Multi-Agent Orchestration)**: 指在 OpenCloud 平台中，通过定义不同 Agent 之间的调用关系和协作流程，使其能够协同工作，共同完成一个端到端复杂任务的能力。
- **Skill (技能)**: Agent 拥有的具体功能或工具，通常以脚本文件（如 Python 脚本）的形式存在，用于执行特定操作，例如抓取热点数据、改写内容或发送消息。
- **定时任务 (Scheduled Task)**: 一种自动化机制，允许系统在预设的时间间隔（例如每小时、每分钟）自动触发某个 Agent 或脚本执行特定任务，以实现实时或周期性的数据处理。
- **Channel (渠道)**: 指消息分发的目标平台，例如飞书、企业微信、Telegram 等。OpenCloud 允许一个“龙虾”（即一个应用实例）配置多个分发渠道。
