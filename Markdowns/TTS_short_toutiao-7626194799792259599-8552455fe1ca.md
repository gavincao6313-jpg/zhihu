# 基于Obsidian + Claude Code构建个人第二大脑

## 1. 内容概览
视频演示了如何利用Obsidian笔记软件和Claude Code AI工具，结合Karpathy的知识库方法，构建个人的第二大脑（AI知识库）。涵盖了软件安装、Skill部署、知识库初始化、Graph View可视化、以及通过Open Web插件和Claude Code浏览器插件快速捕获网页和视频内容并自动整理入库的全过程。

## 2. 时间线

### [00:00:00 - 00:01:00] 引言与工具介绍
*   **目标**：构建个人第二大脑（AI知识库）。
*   **工具组合**：Obsidian（本地笔记，天然适配本地文件，AI读取门槛低） + Claude Code。
*   **优势**：离线数据永远属于自己，支持Markdown，AI可阅读本地文件。
*   **安装**：官网下载Obsidian，打开并创建iCloud文件夹存储笔记。

### [00:01:00 - 00:02:00] 环境准备与Skill安装
*   **原始资料**：放入创建的文件夹。
*   **终端/插件**：打开终端或使用VS Code侧边栏插件运行Claude Code。
*   **安装Marketplace**：复制Karpathy分享的命令添加到Marketplace。
*   **安装Skill**：复制安装命令，Reload生效。
*   **构建方法**：使用Karpathy原始的Skill以及作者分享的特定Skill。

### [00:02:00 - 00:03:00] 构建知识库与Graph View
*   **执行命令**：在Claude Code输入框输入指令，学习Skill并基于Karpathy方法构建完整知识库。
*   **Think指令**：结尾使用彩虹色的`/think`让Code以最高强度思考。
*   **结果展示**：
    *   **Graph View**：文档间的联系以箭头和节点展现（如Code源码分析节点）。
    *   **索引文件**：按类别整理和索引所有文件，记录更改编译查询Link历史。

### [00:03:00 - 00:04:00] 新增资料：Open Web插件
*   **痛点**：收藏文章后不再查看。
*   **解决方案**：安装Open官方插件`open web`。
*   **操作流程**：
    1.  浏览器右上角点击Open Logo。
    2.  插件整理好信息，点击“Add to Obsidian”。
    3.  笔记自动添加到Obsidian文件夹。
    4.  终端告诉Claude Code `ingest`（摄入）这篇笔记到知识库。
    5.  Code读取Skill，创建Markdown文件，按需求整理并放入知识库。

### [00:04:00 - 00:05:00] 视频资料捕获与复杂网页提取
*   **视频处理**：Open插件同样适用。不仅提取基础信息，还提取所有字幕信息（带时间戳）。
*   **复杂网页**：若插件无法很好提取，使用**Claude Code浏览器插件**。
    *   基于指示完成任务。
    *   点击同意计划。
    *   等待操作，提取笔记内容并总结。
*   **总结**：大幅降低整理知识门槛，所有浏览资料可方便加入笔记，AI帮忙整理。
