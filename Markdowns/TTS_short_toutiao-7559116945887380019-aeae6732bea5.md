# 基于 Dify Agent 搭建数据查询 AI 应用：自然语言 SQL 生成与多数据源切换

## 1. 内容概览
本视频演示了如何使用 Dify 平台搭建一个基于 Agent 的自然语言数据查询应用。该应用允许用户通过聊天方式查询多个部门（生产、市场、物资）的数据，支持多维统计（同比、环比等）。核心实现包括：构建包含 SQL 示例和表结构说明的知识库、编写复杂的提示词以引导模型检索表和生成 SQL、将 SQL 执行工作流发布为工具供 Agent 调用。视频对比了 Agent 与工作流模式的优缺点，强调了 Agent 在灵活性和多数据源切换上的优势。
