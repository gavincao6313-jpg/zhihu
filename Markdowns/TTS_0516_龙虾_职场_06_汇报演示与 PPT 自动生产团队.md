# 龙虾_职场_06_汇报演示与 PPT 自动生产团队

## 1. 视频元数据
- **推测主题：** 本视频详细讲解了如何使用OpenCloud平台，通过多Agent编排实现汇报演示与PPT的自动化生成，包括视觉美化、数据报表、情报脱水和PPT架构生成等功能。
- **核心关键词：** OpenCloud, Agent编排, PPT自动生成, 汇报演示, 视觉美化, 数据报表, 情报脱水, Skill, 调度中心, ECharts, 硅基流动, API Key
- **适用受众/场景：** 适用于需要自动化生成报告、演示文稿的个人或企业用户，尤其是对Agent编排、AI工具集成感兴趣的开发者和技术爱好者。

## 2. 核心知识字典（Glossary）

- **Agent编排 (Agent Orchestration):** 指通过定义多个AI Agent（智能体）及其协作流程，实现复杂任务自动化处理的能力。在本案例中，调度中心Agent负责协调视觉Agent、报表渲染Agent、情报脱水Agent和PPT架构师Agent并行或串行工作。
- **Skill (技能):** 在OpenCloud平台中，Skill是Agent执行特定任务所需的功能模块，通常以Python脚本或配置文件形式存在。例如，PPT生成器Skill、ECharts图表生成Skill、图像生成Skill等。
- **调度中心 (Scheduling Center):** 负责接收用户需求，根据预设的调度规则和架构，协调并触发不同的Agent执行任务，并将各Agent的产出进行整合。
- **情报脱水 (Information Dehydration):** 指从原始报告或文本中提取核心信息，去除冗余、重复表达和噪声，进行数据清洗和精炼的过程。
- **硅基流动 (Guiji Liudong):** 一个国内的AI服务聚合平台，提供多种AI模型（如文生图）的API接口，作为本案例中视觉美化Agent的底层能力提供者。
