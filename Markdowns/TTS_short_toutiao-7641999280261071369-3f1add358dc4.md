## 3. 关键事实
*   **项目名称**：cute-claude-hooks
*   **主要功能**：将 Claude Code 的英文界面和命令行输出实时翻译为中文。
*   **技术原理**：通过 Hook 脚本拦截 Claude Code 的执行过程，实现命令的中文解释和界面汉化。
*   **资源规模**：内置翻译字典包含 151+ 个常用词条，且支持用户自定义扩展。
*   **适用场景**：适合不熟悉英文术语的开发者，或希望降低 Claude Code 使用门槛的用户。
*   **成本**：完全免费，开源项目。
