# 唤醒你的 OpenClaw：精选 10 个必装 Skill

## 1. 内容概览
本视频针对用户在使用 OpenClaw 时感觉“什么也干不了”或“笨”的问题，指出原因是未安装合适的 Skill（插件）。基于 GitHub 下载量前三和作者高频使用经验，从 17,000 多个 Skill 中筛选出 10 个核心推荐。这些 Skill 涵盖安全审查、自我进化、搜索、摘要、技能查找、行为约束、前端开发最佳实践、GitHub 操作及浏览器自动化。视频强调正确配置后 Agent 能力可提升至少 3 倍，并详细介绍了每个 Skill 的功能、适用场景及安装建议。

## 2. 时间线

### [00:00:00 - 00:00:15] 背景与问题引入
OpenClaw 本身功能有限，需通过安装 Skill 扩展能力。GitHub 上有 17,000 多个 Skill，经过交叉筛选推荐 10 个核心 Skill，装完后 Agent 能力至少增强 3 倍。

### [00:00:15 - 00:00:30] Skill 1: skill-vetting (安全审查员)
建议在安装其他 Skill 前优先安装。用于扫描安全风险，检查红旗标识、权限范围和可疑模式，检测 Prompt 注入、工具投毒及隐藏代码。

### [00:00:30 - 00:00:51] Skill 2: self-improving-agent (自我进化)
解决大模型“失忆症”。自动将错误日志、成功经验存入本地记忆库。越用越聪明，第 30 天的 Agent 比第一天更强，形成数字分身。

### [00:00:51 - 00:01:04] Skill 3: tavily-search (专属搜索引擎)
让 Agent 具备上网搜索能力。不同于传统引擎返回一堆链接，它返回干净、结构化且对 AI 友好的内容，Agent 拿到即可直接使用。

### [00:01:04 - 00:01:21] Skill 4: summarize (全格式内容摘要专家)
支持网页、PDF、图片、音频、视频等多种格式。日常用法是“搜索 + Summarize”，先搜到页面再压缩成 500 字上下文，效率翻倍。

### [00:01:21 - 00:01:30] Skill 5: find-skills (寻找 Skill)
用自然语言告诉 Agent 需要什么技能，它帮你找到并安装对应的 Skill。不知道装什么时直接问它。

### [00:01:30 - 00:01:46] Skill 6: using-superpowers (行为纪律框架)
核心理念是让 Agent 严格按照“先查 Skill 再行动”的规则做事，杜绝偷懒和无序探索。当发现 Agent 不按规定使用已装 Skill 而瞎搞时，用它来管教。

### [00:01:46 - 00:02:09] Skill 7 & 8: React 最佳实践 & 前端设计
包含 `react-best-practices` (大厂最佳实践，性能优化圣经) 和 `frontend-design` (提升页面设计感)。Anthropic 官方出品，强制 Agent 先做设计思考再写代码，从“能跑就行”提升到“设计合格”。

### [00:02:09 - 00:02:24] Skill 9 & 10: GitHub 操控 & 浏览器操作
`github` 全流程操控，让 Agent 直接操作仓库，ASS PR/RCI 一条命令搞定。`agent-browser` 让 Agent 像人一样打开网页、点击按钮、填写表单、截图。最后展示 Skills 列表总结。

## 3. 关键事实
*   **平台数据**：OpenClaw 拥有 220,000+ Stars，SkillHub 有 17,000+ Skills。
*   **效果量化**：安装推荐的 10 个 Skill 后，Agent 战力至少强 3 倍。
*   **核心痛点**：大模型的“失忆症”需要通过本地记忆库解决；Agent 的“懒惰”和“无序探索”需要通过纪律框架约束。
*   **工作流建议**：推荐组合为 `tavily-search` + `summarize`，将整篇内容压缩为 500 字上下文以提升效率。
*   **开发规范**：前端开发应遵循 Anthropic 官方出品的规范，强调设计思考前置。
