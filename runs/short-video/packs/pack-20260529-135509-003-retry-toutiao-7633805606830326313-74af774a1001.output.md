# 仓库：Claude Code 最佳实践（82条实战经验）

## 1. 内容概览

本视频介绍了一个名为 `claude-code-best-practice` 的 GitHub 仓库，该仓库由 Claude Code 的作者 Boris Cherry 推荐。视频核心内容是讲解该仓库收录的 **82 条实战经验**，这些经验结合了官方推荐的技巧（RIS 所推技巧）和社区验证过的最佳实践。

视频重点拆解了以下关键领域：
*   **上下文管理**：防止“上下文腐烂”，利用独立 Agent 窗口隔离任务。
*   **自动化与定时任务**：使用 `/loop` 在云端执行长期任务，解放本地设备。
*   **多智能体并行**：通过 Agent Teams 实现前端、后端、日志分析的并行工作。
*   **权限与安全**：配置白名单和权限策略，减少不必要的确认打断。
*   **效率工具**：Slash Commands（斜杠命令）、Teleport（远程控制）等高级功能。
