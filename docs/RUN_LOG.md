# 运行日志总账

Date: 2026-05-17

本文档用于记录每日任务运行情况、代码修改前后的运行对比、异常日志处理过程、验证结果和剩余风险。

它是项目共享文件，Windows 和 Mac 都需要遵循同一记录规则。原始 `*.log` 保持本地，不直接提交；需要共享的证据摘录、分析结论和复盘结果写入本文档或 `runs/*.md`。

## 记录原则

1. 每天至少按日期新增一节。
2. 每次生产运行、验证运行、异常中断、重跑，都要留下可回溯记录。
3. 异常日志不能只贴现象，要写清楚判断依据、处理动作、验证结果和是否复发。
4. 涉及代码修改前后对比时，要记录修改前基线、修改后结果和对比结论。
5. 不记录 API key、Cookie、Authorization、签名 URL、完整本机私有路径等敏感信息。
