# WIN 交接：P2-6 质量基线评测（2026-06-04）

**目标：** 对「AI落地的胜负手」这场直播，量化**分块流式处理**和**全文件处理**两种模式的产出质量差异。  
**Mac 已完成：** `Markdowns/TTS_0604_replay-xiaoe-20260603.md`（小鹅通回放 URL · 分块模式 · Gemini 合成）

> **关键说明：** 回放视频下载到本地后和本地 MP4 **走完全相同的工作流**，无任何区别。
> 真正有意义的对比只有一个维度：
> - **分块流式模式**（`zhihuTTS_stream.py`）：URL 按 60 秒切片，每块独立转写，块间拼接——直播和回放流都走这条路
> - **全文件模式**（`zhihuTTS_video.py`）：完整 MP4 一次性提取帧和转写，上下文最完整

| 路径 | 处理模式 | 入口 | 期望产物 |

## 路径B：分块流式处理（回放 URL，60s 切片）

### 前提
- 已有 `xiaoe_auth_state.json`（小鹅通登录态）
- 设置了 `DASHSCOPE_API_KEY`

### Step 1 — 截取回放流地址
```powershell
set XIAOE_PAGE_URL=<昨晚直播的小鹅通回放页面URL>

python probe_xiaoe_stream.py %XIAOE_PAGE_URL% xiaoe_auth_state.json
```
成功后终端会打印 `M3U8_URL=https://...`，复制这个 URL。

### Step 2 — 跑回放流 pipeline（分块转写 + Qwen 合成）
```powershell
set STREAM_URL=<上一步拿到的M3U8_URL>
set DASHSCOPE_API_KEY=your_key

python zhihuTTS_stream.py ^
  --url "%STREAM_URL%" ^
  --base replay-xiaoe-20260603-win ^
  --duration 0 ^
  --chunk-duration 60 ^
  --provider qwen ^
  --qwen
```
> `--duration 0` = 处理到流结束为止。预计耗时 60~90 分钟（转写 + 合成）。

### 期望产物
```
runs/stream-replay-xiaoe-20260603-win/

  manifest.json
  final-qc.json          ← 质量指标在这里

  TTS_stream-replay-xiaoe-20260603-win-qwen.md   ← 主产物
```

## 路径C：全文件处理（完整 MP4，一次性提取）

### 前提
- Windows 本地有昨晚直播的 MP4（`Videos/` 目录下，或自行指定路径）
- 确认文件名，下面用 `VIDEO_NAME` 指代（不带 `.mp4`）

### Step 1 — 提取关键帧 + 转写
```powershell
set VIDEO_NAME=AI落地的胜负手-20260603

python zhihuTTS_video.py --video "Videos\%VIDEO_NAME%.mp4"
```
产出 `cache/payload/%VIDEO_NAME%.payload.json`（帧 + 逐字稿打包）。

### Step 2 — Qwen 合成
```powershell
set DASHSCOPE_API_KEY=your_key

run_replay_qwen.bat "cache\payload\%VIDEO_NAME%.payload.json" local-20260603
```
产出 `Markdowns/TTS_stream-local-20260603-qwen.md`。

## 异常处理

| 现象 | 处理 |
|------|------|
| `probe_xiaoe_stream.py` 超时 | 检查 `xiaoe_auth_state.json` 是否过期，重新 `python save_xiaoe_auth.py` 登录 |
| `zhihuTTS_stream.py` 中途失败 | 无 `--resume`，重新跑；已完成 chunk 产物不会被删除 |
| `run_replay_qwen.bat` 合成失败 | 用脚本注释里的续跑命令加 `--resume-window-notes` 重试 |
| MP4 文件找不到 | `zhihuTTS_video.py --video` 接绝对路径也可以 |
| `final-qc.json` 不存在 | 合成尚未完成或失败，检查终端报错 |
